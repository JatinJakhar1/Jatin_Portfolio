# 🌐 Jatin Portfolio

A modern and responsive **personal portfolio website** built using **HTML, CSS, and JavaScript** to showcase my projects, skills, and experience.

---

## 🚀 Live Demo
https://jatinjakhar1.github.io/Jatin_Portfolio/

---

## 📌 Overview

This portfolio website represents my journey as a developer and highlights my work in **web development and AI projects**.
It is designed with a clean UI, smooth animations, and an intuitive user experience.

---

## ✨ Features

* 🎯 Responsive design (Mobile + Desktop)
* 💻 Modern and clean UI
* ⚡ Smooth scrolling and animations
* 🧠 Skills and technology showcase
* 🎓 Education timeline section
* 📂 Project showcase with live/demo links
* 📝 Blog section with interactive article view
* 📞 Contact section with WhatsApp & Email integration

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript (ES6)** – Interactivity

## ⚙️ Getting Started

### 🔹 Option 1: Open directly

Open `index.html` in your browser.

### 🔹 Option 2: Run using local server

```bash
python -m http.server 8000
```

Then open:
👉 http://localhost:8000

---

## 🎨 Customization

To personalize your portfolio:

* Edit sections in `index.html`
* Modify styles in `style.css`
* Update interactions in `script.js`
* Replace images inside `assets/`
